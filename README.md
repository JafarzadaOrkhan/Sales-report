This project presents an interactive **Sales Report Dashboard** designed to analyze sales performance across different regions and time periods.

## Overview
The dashboard provides key business insights such as:
- Total Sales: **$110M**
- Number of Sales Agents: **3,806**
- Sales by Territory, Country, and Continent
- Customer Distribution by Region
- Sales Trend Over Time

## Key Insights
- **Southwest** and **North America** generate the highest sales.
- Customer distribution varies significantly across regions.
- Sales show a steady increase over time, followed by a slight decline.

## Tools & Technologies
- Power BI (Data Modeling & Visualization)
- Interactive filters (Year, Continent)

## Purpose
This dashboard is built to support data-driven decision-making by providing a clear and concise view of sales performance.
